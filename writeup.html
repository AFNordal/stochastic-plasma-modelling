Analysis of filtered poisson process of double exponentials

This writeup presents the most important results. For technical reference, see readme.md.

Data generation

The time series, Φ(τ)\Phi(\tau)Φ(τ) was generated with the following conditions:

  • τd=1μs\tau_d=1 \mu sτd​=1μs
  • Uniformly distributed arrival times
  • Exponentially distributed amplitudes with scale 1
  • Predetermined number of events; n=γTn=\gamma Tn=γT
  • Four different values of γ\gammaγ; 0.1,1,100.1, 1, 100.1,1,10 and ∞\infty∞.
  • For each γ\gammaγ, six different values of λ\lambdaλ; 0,0.1,0.2,0.3,0.40, 0.1, 0.2, 0.3, 0.40,0.1,0.2,0.3,0.4 and 0.50.50.5
  • A total duration of T=100000μsT=100000\mu sT=100000μs
  • dt=0.01μsdt=0.01\mu sdt=0.01μs

The realizations with real numbered γ\gammaγ were generated using the superposed pulses package. The one with γ=∞\gamma=\inftyγ=∞ was generated by convolving the pulse shape ϕ(τ)\phi(\tau)ϕ(τ) with a series of normally distributed amplitudes.

For each value pair {γ,λ}\{\gamma, \lambda\}{γ,λ}, 100 realizations of the series were made.

Conditional averaging

The conditional averaging algorithm used is a modified version of from the one supplied in the fpp-analysis-tools package. It was modified to support normalizing the amplitudes of the conditional events as well as plotting an illustration of the analysis process. The base conditions used in the analysis were:

  • Size of conditional events, Δ=3μs\Delta=3\mu sΔ=3μs
  • Threshold of 2.5Φrms2.5\Phi_{rms}2.5Φrms​
  • Do not allow overlapping conditional events (window=True)
  • Normalize the amplitudes of each conditional event

Fit analysis

A least squares fit was performed to extract shape parameters from the conditionally averaged waveforms. Three different variations of this method were tested. First of all, two different functions were used; One double exponential, where the constant term and scaling factor were shared between the rising and falling flank, and one with both flanks having an independent constant term and scaling factor. Both of these were tested using the entire conditional waveform, τ∈[−Δ/2,  Δ/2]\tau\in[-\Delta/2,\;\Delta/2]τ∈[−Δ/2,Δ/2]. Furthermore, the function with shared parameters was tested with a domain of τ∈[−τr,  τf]\tau\in[-\tau_r,\;\tau_f]τ∈[−τr​,τf​], where τr\tau_rτr​ and τf\tau_fτf​ were estimated iteratively: First, the function was fitted for τ∈[−Δ/2,  Δ/2]\tau\in[-\Delta/2,\;\Delta/2]τ∈[−Δ/2,Δ/2], from which τr,i\tau_{r, i}τr,i​ and τf,i\tau_{f, i}τf,i​ were estimated. Then, τr,i+1\tau_{r, i+1}τr,i+1​ and τf,i+1\tau_{f, i+1}τf,i+1​ were estimated on a domain of [τr,i,  τf,i][\tau_{r, i},\;\tau_{f, i}][τr,i​,τf,i​] until τd,i=τr,i+τf,i\tau_{d, i}=\tau_{r, i}+\tau_{f, i}τd,i​=τr,i​+τf,i​ converged. The vast majority of estimates converged within 100 iterations.

All three methods were tested on the base conditional waveform for each 100 realizations for each {γ,λ}\{\gamma, \lambda\}{γ,λ} pair. The mean estimates of λ\lambdaλ and τd\tau_dτd​ with related standard variations are shown in the below plot, along with the true value in black:

The first takeaway is that all methods give results within the same order of magnitude, and they all suggest the same trend (except for λ=0\lambda=0λ=0); The slope the estiamte curve of λ\lambdaλ decreases with higher γ\gammaγ, and the slope of the τd\tau_dτd​ estimate increases, both giving less accurate estimates for higher γ\gammaγ.

We can also see that the method with shared parameters and constant domain ("connected") gives overall best performance for estimates of both parameters.

The method with shared parameters and iteratively estimated domain ("connected iterated") significantly underperforms in estimating τd\tau_dτd​ for higher γ\gammaγ.

The method with independent parameters and constant domain ("disconnected") underperforms in estimating both parameters for λ=0\lambda=0λ=0. This is due to the rising flank being too short, only consisting of 2-3 data points. Therefore the floor curve leading up to the flank, consisting of a lot more data points, is estimated instead of the actual flank.
